# Log Analyzer (IST)

This project simulates and analyzes user activity logs in **Indian Standard Time (IST)**.  
It records familiar actions such as attending class, using AI for classwork, submitting assignments, and using other applications.

## Features
- Generates logs in a call-log style format
- Tracks time spent on each activity
- Provides trend analysis and insights
- Visualizes results with bar and pie charts
